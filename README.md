- [X] 世界名称：
  - [X] 导弹战争：🗡
  - [X] 资源世界：⛏
- [X] 服务器负荷emoji：
  - [X] TPS>=18：&a🌑
  - [X] TPS>=15：&6🌑
  - [X] TPS>=10：&c🌑
  - [X] TPS<10：&4🌑
- [X] 世界时间emoji：
  - [X] 0~12000 ticks：&6☀
  - [X] 其它：&e🌚
